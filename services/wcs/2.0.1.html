# WCS 2.0.1

WCS 2.0.1提供如下操作:

  • GetCapabitities：返回服务级元数据，它是对服务信息内容和要求参数的一种描述。

  • DescribeCoverage：支持用户从特定WCS服务器获取一个或多个覆盖的详细的描述文档。

  • GetCoverage：可根据查询要求返回一个包含或者引用被请求的覆盖数据的响应文档。

参考文档:

09-110r4_WCS_Core_2.0.1.pdf(http://docs.opengeospatial.org/is/17-089r1/17-089r1.html)

12-100r1_OGC_GML_Application_Schema_-Coverages-_GeoTIFF_Coverage_Encoding_Profile.pdf(https://portal.ogc.org/files/?artifact_id=54813)

# GetCapabilities

# 请求

请求参数

参数名称 是否必填 描述
service 是 服务名:WCS
request 是 请求名称:GetCapabilities
version 否 版本:2.0.1
acceptVersions 否 请求的 WCS 服务的版本号序列，各个版本间用逗号隔开，并把最希望返回的版本的服务写在最前面。
sections 否 请求返回的服务端元信息的部分（Sections），多个部分之间用逗号隔开。
updateSequence 否 服务元信息文档的版本号，因此号码是一个增量，每次修改会增加一个值。
如果此参数缺省，或参数值不是服务端支持的版本号，则返回最新版本的服务元信息。

请求示例

{{server}}/igs/rest/services/:serviceName/WCSServer?version=2.0.1&request=GetCapabilities&service=wcs
1

# GetCoverage

# 请求

请求参数

参数名称 是否必填 描述
service 是 服务名
示例：WCS
request 是 请求名
示例：GetCoverage
version 否 版本
示例：2.0.1
coverageId 是 指定所要请求的图层，从GetCapabilities返回的XML信息的CoverageId节点中获取。
mediaType 否 同时需要XML和图像输出时，应该设置为' multipart/related '(这是该参数目前唯一可能的值)，IGServer暂不支持该参数。
format 否 返回的格式，从GetCapabilities返回的XML信息的formatSupported节点中获取。
示例：format=image/tiff
subset 否 允许使用给定的范围来获取数据，参数包含轴类型(宽高)和给定轴的范围信息(宽高的范围区间)，每个轴唯一。
示例：subset=Long(0,10)&subset=Lat(0,10)，Long表示宽，Lat表示高。
subsettingCrs 否 范围参照系，传入的subset范围信息必须是该参照系下的范围。
示例：subsettingCrs=EPSG:4326
outputCrs 否 输出参照系，数据的参照系通过此参数设置，如果数据本身参照系与输出参照系不同，会返回投影后的数据。
示例：outputCrs=EPSG:4326
scaleFactor 否 使用这个参数(一个正浮点数)可以调整输出图像的大小，图像的宽高等比例放大或缩小
示例：scaleFactor=1.5
scaleAxes 否 指定轴类型(宽高)进行放大或缩小。
示例：scaleAxes=Long(2)&scaleAxes=Lat(3)，Long表示宽，Lat表示高。
scaleSize 否 指定轴类型(宽高)的像素大小。
示例：scaleSize=Long(100)&scaleSize=Lat(100)，Long表示宽，Lat表示高。
interpolation 否 重新缩放图像的插值方法。从GetCapabilities返回的XML信息的interpolationSupported节点中获取。
示例：interpolation=http://www.opengis.net/def/interpolation/OGC/1/nearest-neighbor
rangeSubset 否 选择波段,从DescribeCoverage返回的XML信息的field节点中获取。
示例:rangeSubset=band1(单波段)或者rangeSubset=band3,band2,band1(多波段)
geotiff:comperssion 否 压缩方式。
示例：geotiff:comperssion=Jepg或者LZW
geotiff:jpeg_quality 否 Jpeg压缩质量。
示例：geotiff:jpeg_quality=100

请求示例

{{server}}/igs/rest/services/:serviceName/WCSServer?version=2.0.1&request=GetCoverage&service=wcs&coverageId=world__world&format=image/png
1

# 响应

结果根据传入的格式返回图片、tiff数据、text文本

# 示例

1.默认请求(返回全图)

{{server}}/igs/rest/services/:serviceName/WCSServer?version=2.0.1&request=GetCoverage&service=wcs&coverageId=world__world&format=image/png
1

2.根据范围请求

{{server}}/igs/rest/services/:serviceName/WCSServer?version=2.0.1&request=GetCoverage&service=wcs&coverageId=world__world&format=image/png&subset=Long(-10,10)&subset=Lat(-10,10)
1

3.指定宽高

{{server}}/igs/rest/services/:serviceName/WCSServer?version=2.0.1&request=GetCoverage&service=wcs&coverageId=world__world&format=image/png&subset=Long(-10,10)&subset=Lat(-10,10)&scaleSize=Long(100)&scaleSize=Lat(100)
1

4.缩放返回的图片

{{server}}/igs/rest/services/:serviceName/WCSServer?version=2.0.1&request=GetCoverage&service=wcs&coverageId=world__world&format=image/png&subset=Long(-10,10)&subset=Lat(-10,10)&scaleFactor=2
1
