# 概述

# 服务描述

Web Map Service，网络地图服务（也称为 动态地图服务），它是利用具有地理空间位置信息的数据制作地图，其中将地图定义为地理数据的可视化表现，能够根据用户的请求，返回相应的地图，包括PNG、GIF、JPEG等栅格形式，或者SVG或者WEB CGM等矢量形式。WMS支持HTTP协议，所支持的操作是由URL决定的。

# 支持版本

IGServer目前支持的WMS版本包括

  • 1.1.1
  • 1.3.0

# 规范

# 坐标顺序

坐标顺序与服务的版本号、坐标系类型有关，OGC没有明确规定该规则，IGServer参考了GeoServer、ArcGIS Server等GIS服务器产品。

WMS 涉及坐标顺序的场景包含：

  • GetCapabitities返回文档中的BoundingBox标签，表示描述图层的范围，包含xmin、ymin、xmax、ymax、SRS(1.1.1)或者CRS(1.3.0)属性

  • GetMap传入bbox参数，表示传入取图的范围，格式一般为xmin,ymin,xmax,ymax

在 IGServer 提供的 WMS 服务中，上述场景的坐标顺序存在需要变换的情况，具体情况如下：

经纬度坐标系4326, 4214, 4610, 4490 高斯坐标系[2327,2442], [4491,4554], [21413,21423],[21473,21483] Web墨卡托投影坐标系3857
1.1.1 XY XY XY
1.3.0 YX YX XY
