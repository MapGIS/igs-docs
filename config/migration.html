# 服务实例迁移v1->v2对应关系

服务实例v1 服务实例v2
数据源 服务类型 数据源 服务类型
二维地图文档 二维地图文档服务、WMS、WFS 地图文档 地图服务（主服务、可支持动态瓦片）、要素服务、WMS服务（如果勾选）、WFS服务（如果勾选）、WMTS服务（如果勾选）
瓦片数据 瓦片服务、WMTS 瓦片文件、MongoDB 瓦片服务（主服务）、WMTS服务（如果勾选）
目录文档、目录瓦片 目录文档服务、目录瓦片服务 文件夹 同二维地图文档、瓦片服务
矢量瓦片数据 矢量瓦片服务 矢量瓦片文件、文件夹、MongoDB 矢量瓦片服务
三维地图文档服务 G3D服务 地图文档 场景服务
三维MongoDB缓存服务 G3D服务 暂无 场景服务（只能使用mapx套mongodb）
GIS图层 WMS、WFS GIS图层 WMS、WFS

# 服务实例v2对应REST v1接口支持

REST v1接口 服务实例v2
MRMS/MRFS/MRCS 二维地图文档数据源的服务、瓦片数据源的服务
MRFWS 工作流服务
G3D 三维地图文档数据源的服务
WMS（文档） 二维地图文档数据源的WMS服务
WMS（图层） 图层数据源的WMS服务
WFS（文档） 二维地图文档数据源的WFS服务
WFS（图层） 图层数据源的WFS服务
WMTS 瓦片数据源的WMTS服务
